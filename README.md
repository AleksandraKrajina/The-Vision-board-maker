# Vision Board Maker (Vue + Laravel)

Project is split into:

- `frontend/` → Vue 3 app (browser ESM build)
- `backend/` → Laravel API-oriented structure

## Frontend run

You can run frontend statically:

```bash
cd frontend
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173`.
